## 行為守則

作為社區的一份子，我們承諾打造一個自由開放和氣氛舒適的交流環境，並積極參與項目推廣和發展，為所有人提供一個良好的社區體驗。

每位參與項目的人 (包括線上和線下活動) 都必須同意 ProtoSchool [行為守則](./CODE_OF_CONDUCT.md).
